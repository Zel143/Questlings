# Questling Type & Habit Mapping

This document formalizes the association between Questling types and habit categories to ensure thematic consistency.

| Habit Category | Questling Type | Core Philosophy | Example Species |
| :--- | :--- | :--- | :--- |
| **Hydration** | Water | Flow, purity, and refreshment. | Aqualon, Droplet, Mistral |
| **Fitness** | Earth / Physical | Strength, endurance, and grounding. | Bouldron, Thumper, Terraroo |
| **Study / Work** | Electric / Logic | Focus, energy, and mental sparks. | Circuit, Glint, Voltwing |
| **Mindfulness** | Nature / Spirit | Calm, growth, and inner peace. | Wispy, Bloom, Zenleaf |
| **Social / Co-op** | Light / Heart | Connection, support, and friendship. | Gloon, Hearty, Solis |
| **Chores / General** | Fire / Action | Initiative, energy, and "burning" through tasks. | Burny, Spark, Kindle |

## Implementation Impact
When a user creates a habit in a specific category, the game will suggest or default to an Egg of the corresponding type. Earning a Questling of a specific type "buffs" the user's efficiency or rewards when performing habits in that category.
